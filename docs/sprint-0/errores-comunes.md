# Errores comunes — Sprint 0

Guía rápida para la puesta en marcha del repo y el tablero.

## No aparece la invitación a GitHub Classroom

- Revisar spam de correo y que todos usen el mail correcto.
- Si alguien entra con otra cuenta de GitHub, no verá el mismo repo.

## El README no cumple lo pedido

- Falta nombre del equipo, integrantes con link a GitHub, ficha del cliente o link al **GitHub Projects**.
- Solución: seguir el enunciado del Sprint 0 punto por punto.

## Tablero con columnas distintas a las acordadas

- El manual pide columnas tipo backlog / trabajo en curso / hecho (según lo definido en cursada).
- Solución: renombrar columnas para alinearlas con el flujo docente.

## Milestones sin fecha o sin descripción

- Sin fecha es difícil planificar entregas.
- Solución: completar título, fecha de cierre y objetivo del sprint en cada Milestone.
