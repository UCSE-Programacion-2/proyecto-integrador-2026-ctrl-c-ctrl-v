# Sprint Review - Guia unificada

Esta carpeta centraliza los recursos para estandarizar el entregable de Sprint Review en video en todos los sprints.

## Recursos disponibles

1. **Plantilla para equipos**
   - Archivo: **[sprint-review-template.md](./sprint-review-template.md)**
   - Uso: guion base para estructurar el video de forma profesional.

2. **Rubrica de evaluacion**
   - Archivo: **[sprint-review-rubrica.md](./sprint-review-rubrica.md)**
   - Uso: criterios y puntaje para correccion homogenea.

3. **Checklist para estudiantes**
   - Archivo: **[sprint-review-checklist-estudiantes.md](./sprint-review-checklist-estudiantes.md)**
   - Uso: control final antes de enviar el video.

4. **Checklist para docentes**
   - Archivo: **[sprint-review-checklist-docentes.md](./sprint-review-checklist-docentes.md)**
   - Uso: revision rapida y consistente de cada entrega.

## Flujo recomendado

1. El equipo prepara el video con la plantilla.
2. Verifica la entrega con el checklist de estudiantes.
3. El docente corrige con la rubrica y checklist docente.
