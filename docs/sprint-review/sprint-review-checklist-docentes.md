# Checklist - Sprint Review (docentes)

Guia rapida para revisar videos de Sprint Review de forma homogenea.

## Validaciones iniciales

- [ ] El video corresponde al sprint evaluado.
- [ ] Participan todos los integrantes.
- [ ] Se identifican roles PO y SM.
- [ ] Existe evidencia concreta del trabajo del equipo.

## Revision de contenido

- [ ] Se compara plan inicial vs resultado real.
- [ ] Cada integrante presenta su aporte individual.
- [ ] Se demuestra incremento funcional acorde al sprint.
- [ ] El equipo explica aciertos y errores con honestidad.
- [ ] Hay plan de mejora accionable para el siguiente sprint.

## Criterios de calidad

- [ ] El relato es claro, ordenado y profesional.
- [ ] Las evidencias mostradas son verificables.
- [ ] Lo expuesto coincide con repo/tablero/PRs.
- [ ] Se observa evolucion de trabajo en equipo y uso de Scrum.

## Cierre

- [ ] Puntaje asignado segun `docs/sprint-review/sprint-review-rubrica.md`.
- [ ] Feedback breve: fortalezas (2) y mejoras (2).
